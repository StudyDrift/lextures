# SS — Screen Sharing (Cableless Classroom Presentation)

Net-new capability: let a student or instructor share their **entire screen** from a browser to the
classroom display (and optionally to classmates' devices) in one click — replacing the HDMI/VGA cable,
dongle, and "why is it blank?" dance. Sub-second latency, in-house Selective Forwarding Unit (SFU), behind
a platform + per-course feature flag. Self-hostable; no external SaaS in the critical path.

# Module 2 summary

- Variables name values; `=` assigns, `==` compares.
- Core types: `int`, `float`, `str`, `bool`.
- Convert with `int()`, `float()`, `str()`.
- `input()` returns a **string** — convert before numeric comparisons.
- f-strings: `f"Hello, {name}"`.

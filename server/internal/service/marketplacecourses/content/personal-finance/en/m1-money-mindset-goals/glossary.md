# Glossary of key terms

Return here when later modules reuse a term. Definitions are plain-language and aligned with government/nonprofit education materials.

| Term | Plain-language definition |
|---|---|
| **Need** | Spending required for health, safety, or essential obligations. |
| **Want** | Optional spending that adds comfort or enjoyment. |
| **SMART goal** | Specific, Measurable, Achievable, Relevant, Time-bound goal. |
| **MyMoney Five** | Federal framework: Earn, Save & Invest, Protect, Spend, Borrow. |
| **Budget** | A plan for income and expenses over a period (often a month). |
| **Cash flow** | Money in vs. money out over time. |
| **Emergency fund** | Savings set aside for unexpected expenses or income loss. |
| **APY** | Annual Percentage Yield — effective yearly return including compounding. |
| **Compound interest** | Interest earned on both principal and prior interest. |
| **FDIC insurance** | Deposit insurance for covered accounts at FDIC-insured banks (standard limit applies). |
| **Credit report** | Record of credit accounts and payment history. |
| **Credit score** | Number summarizing credit risk based on report data. |
| **APR** | Annual Percentage Rate — yearly cost of borrowing. |
| **Debt avalanche** | Extra payments toward highest-interest debt first. |
| **Debt snowball** | Extra payments toward smallest balance first. |
| **Diversification** | Spreading investments to reduce reliance on any single outcome. |
| **Index fund** | Fund designed to track a market index at relatively low cost. |
| **Tax-advantaged account** | Account with special tax treatment (e.g., many retirement accounts). |
| **Withholding** | Money held from pay for taxes. |
| **Identity theft** | Using someone else's personal information without permission for fraud. |

Sources include [MyMoney.gov](https://www.mymoney.gov/), [CFPB](https://www.consumerfinance.gov/), [Investor.gov](https://www.investor.gov/), and [FDIC](https://www.fdic.gov/consumer-resource-center/money-smart-adults).

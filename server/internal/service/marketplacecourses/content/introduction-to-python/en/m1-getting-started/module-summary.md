# Module 1 summary

- A **program** is a sequence of instructions; this course uses **Python 3**.
- You can run code in a **browser REPL** (zero install) or with a **local** install from [python.org](https://www.python.org/).
- The **REPL** is for quick experiments; **`.py` scripts** are for saved programs.
- `print()` displays values; `#` starts a comment.
- Do not paste secrets into online interpreters.

**Next:** variables, types, `input()`, and f-strings.

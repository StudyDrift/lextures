# Supervised, unsupervised, and reinforcement learning

## Supervised learning

Learn a mapping from inputs to known labels (classify images; predict house prices). Needs labeled examples.

## Unsupervised learning

Find structure without labels (cluster customers; compress data). Useful for exploration and representation learning.

## Reinforcement learning

Learn by trial and error with rewards (game playing; some robotics). The system tries actions and improves policies that earn more reward.

## Overfitting (intuition)

**Overfitting** means the model memorizes quirks of the training set and performs worse on new data. Holding out a test set and preferring simpler patterns when possible are classic defenses — covered in intro materials like [Elements of AI](https://www.elementsofai.com/) and [Google ML Crash Course](https://developers.google.com/machine-learning/crash-course).

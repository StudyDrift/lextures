# Module 6 summary

- `def` defines functions; `return` sends a value back
- Parameters, defaults, and docstrings make functions reusable
- `import` loads standard-library modules such as `math`, `random`, `statistics`
- Prefer return values for reuse; print at the edges of your program

**Next:** string methods, files, and `try`/`except`.

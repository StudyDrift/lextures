# Where to go next

You now have a beginner foundation: run Python, types, control flow, collections, functions, files, and errors.

## Free next steps

- Keep practicing with [Python for Everybody](https://www.py4e.com/) exercises
- Try [CS50P](https://cs50.harvard.edu/python/) for another structured path
- Automate small personal tasks with ideas from [Automate the Boring Stuff](https://automatetheboringstuff.com/)
- Read more of the [official tutorial](https://docs.python.org/3/tutorial/index.html) and [library reference](https://docs.python.org/3/library/index.html)

## Not in this course (on purpose)

Object-oriented design, decorators, async, web frameworks, and data science libraries are follow-on topics — look for future “Automate with Python” / “Python for Data” style courses once you are comfortable with the basics.

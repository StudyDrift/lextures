# Module 2 summary

- Models learn from **data** and **features**; poor data yields poor behavior.
- **Training** fits the model; **inference** uses it on new inputs.
- Know the three learning styles: supervised, unsupervised, reinforcement.
- **Overfitting** is fitting the training set too tightly at the expense of new data.

# Run Python online (zero install)

Before any syntax deep-dive, get a working place to run code.

## Recommended: Python.org shell

Open the free [Python.org interactive shell](https://www.python.org/shell/) in your browser. You should see a prompt like `>>>`.

Type this and press Enter:

```python
print("Hello, Python!")
```

**Expected output:**

```text
Hello, Python!
```

## Alternatives

If that shell is unavailable, use another free online Python 3 REPL (search for “online Python interpreter”). Prefer tools that clearly run **Python 3**, not Python 2.

## What you just did

You ran a **statement**. `print(...)` asks Python to display a value. Quotes mark a **string** (text).

You can keep using an online interpreter for the whole course. Local install is optional and covered next.

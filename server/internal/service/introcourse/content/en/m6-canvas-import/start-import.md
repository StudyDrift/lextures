## Start a Canvas import

1. Go to your **Courses** page.
2. Open **Import from Canvas** (or the import option in the course menu).
3. Enter your Canvas base URL and access token when prompted.
4. Select the courses to import and which content types to include.

[Open your courses page](/courses) to start an import when you are ready.

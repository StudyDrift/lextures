# Module 7 summary

- String methods return new strings (`strip`, `split`, `join`, `replace`, …)
- Read/write text with `open` and prefer `with` + `encoding="utf-8"`
- Use `try` / `except` for expected failures like `FileNotFoundError`
- Read the exception type in error messages — it usually points to the fix

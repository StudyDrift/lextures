## Migrating from Canvas

If you used **Canvas LMS** before, you can import existing courses into Lextures instead of rebuilding everything by hand.

Import when:

- You are moving a term's content to Lextures
- You want modules, pages, and assessments as a starting point
- Your institution supports Canvas API access

If Canvas is not configured yet, the import wizard will guide you through credentials setup on the Courses page.

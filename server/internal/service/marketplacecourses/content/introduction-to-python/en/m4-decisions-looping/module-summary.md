# Module 4 summary

- `if` / `elif` / `else` choose paths; indentation defines blocks
- `while` repeats while a condition is true; `for` iterates over a sequence
- `range(n)` yields `0 .. n-1`; `range(a, b)` yields `a .. b-1`
- `break` exits; `continue` skips to the next iteration

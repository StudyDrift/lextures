# What is a program?

A **program** is a sequence of instructions a computer can follow. You write those instructions in a programming language — here, **Python 3**.

Python reads your code from top to bottom (with loops and functions changing the path). Each line does a small job: print text, store a value, ask a question, or make a decision.

## Why Python?

Python is a common first language: readable syntax, a large standard library, and wide use in data, automation, and AI. This course targets **Python 3.x** and the [official tutorial](https://docs.python.org/3/tutorial/index.html).

## What you will build toward

By the end of the course you will write a small complete program (for example a number-guessing game) using variables, decisions, loops, functions, and files — all with the standard library only.

## Privacy note for online tools

If you use a browser-based interpreter, treat it like a public notepad: **do not paste passwords, API keys, or personal data**.

# Module 3 summary

- Arithmetic: `+ - * / // % **`
- Comparisons return `bool`; combine with `and` / `or` / `not`
- Precedence: `*` before `+`; use `()` to force order
- Strings: `+` concatenates, `*` repeats, `len()` counts characters
- Convert types before mixing `str` and numbers
